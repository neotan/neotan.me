## 1. GENERAL

1. [ ] Is it a greenfield project?
1. [ ] Who is the team? 
1. [ ] What would the first 30 days look like for me?

## 2. COMMUNICATION

1. [ ] Real-Time? 
    * Slack, MS Teams, Discord
1. [ ] Task Manager
    * JIRA, Trello 
1. [ ] Pull Requests
    * GitHub, GitLab, Bitbucket
1. [ ] Email
1. [ ] Meetings
    * Stand-ups
    * Frequency

## 3. WORKFLOW

1. [ ] Do we do Kanban, Scrum, or Power Yoga?
1. [ ] Do we do code reviews?
1. [ ] Do we do feature branches?

## 4. API

1. [ ] What kind of API does your backend offer?
1. [ ] Where do I find the API documentation?

## 5. UI/UX

1. [ ] Should I make it look good or pixel perfect?⚠
1. [ ] Where do I get the mockups from? 
1. [ ] What color scheme, fonts, and assets should I use? 
1. [ ] Are we using in-house icons?

## 6. FRONTEND

1. [ ] Do you have already plans of which libraries you want to use?
1. [ ] How much customization is needed?
1. [ ] Are we using a state management library?
1. [ ] How do we fetch data?
    * React-Query
1. [ ] Should we use TypeScript?
1. [ ] How much should be tested?⚠
